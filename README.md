# Moltshop Website

Landing page for [moltshop.app](https://moltshop.app)

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Static export for Cloudflare Pages:
```bash
npm run build
# Deploy the `out` directory
```
